## Getting Started

1. Clone this repository:
```bash
git clone https://github.com/bxxmzilla1/test-Sam.git
```

2. Open `index.html` in your web browser

That's it! No build process or dependencies required.

## Usage

1. Enter a title and content for your note
2. Click "Save Note" or press `Ctrl+Enter`
3. Your notes will appear in the grid below
4. Click "Edit" to modify a note
5. Click "Delete" to remove a note (with confirmation)
